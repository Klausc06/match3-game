# 模块测试工作流

// turbo-all

## 1. 运行测试门禁
```bash
npm run prompt:check && npm run validate:levels && npm test
```

## 2. 检查网页端表现
使用浏览器子智能体或提示用户打开本地测试服务器，检查以下内容：

### 基础检查项
- [ ] 页面加载无 Console 报错
- [ ] Canvas 正确渲染棋盘
- [ ] 图块加载图片资源正确 (非 Emoji)

### 交互检查项
- [ ] 点击图块有选中高亮
- [ ] 滑动/点击交换可以触发
- [ ] 无效交换被弹回
- [ ] 交换如果触发道具或匹配成功，则执行消除
- [ ] 发呆3秒后游戏能给出 Hint 提示
- [ ] 匹配消除动画正常
- [ ] 下落动画正常
- [ ] 新图块从顶部填充

### 游戏系统检查项
- [ ] 倒计时 90 秒显示并倒数
- [ ] 分数正确累加
- [ ] 连消倍数正确应用
- [ ] 时间结束弹出结算
- [ ] 排行榜正确保存/显示
